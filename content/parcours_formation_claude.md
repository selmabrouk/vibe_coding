Ratio maintenu : **30% theorie  |  70% labs & projets pratiques**

**PHASE 1  Fondations**     Maintenant -> 15 juin  |  2-3h/semaine     ~12 heures

| Type | Module & Ressource | Dur?e |
|---|---|---|
| Theorie | Claude 101  ? lien
Comprendre Claude, ses capacites, patterns d'usage de base. Videos officielles + quiz. | ~2h |
| Lab | AI Fluency ? Framework & Foundations  ? lien
Exercices pratiques : methode 4D, structurer ses prompts, cas d'usage reels. Faire tous les labs. | ~3h |
| Lab | Prompt Engineering intensif  ? lien
S'entrainer sur claude.ai : system prompts, few-shot, chain-of-thought. Appliquer sur StatusPulse. | ~3h |
| Theorie | AI Capabilities & Limitations  ? lien
Comprendre les limites de Claude, hallucinations, cas d'echec. Cours court + quiz. | ~1h |
| Projet | Mini-projet : Assistant StatusPulse
Creer un prompt systeme pour analyser des incidents et generer des rapports SMS (format INC OPEN/CLOSE). | ~3h |

Checkpoint mi-juin  **Certificat AI Fluency obtenu + maitrise des prompts avances**

**PHASE 2  Intermediaire**     15 juin -> 30 juin  |  4-5h/semaine     ~18 heures

| Type | Module & Ressource | Dur?e |
|---|---|---|
| Theorie | Building with the Claude API  ? lien
Architecture Messages API, streaming, context management, gestion des erreurs, modeles Opus/Sonnet/Haiku. Cours phare +8h. | ~5h |
| Lab | API Labs : appels reels depuis Node.js/Express  ? lien
Coder ses premiers appels API Claude. Integrer dans le backend StatusPulse (JWT + roles Admin/Operator/Reader). | ~5h |
| Theorie | Introduction au MCP (Model Context Protocol)  ? lien
Comprendre comment brancher Claude a des outils externes : DB, APIs, fichiers. Cours officiel Academy. | ~2h |
| Lab | MCP Lab : connecter Claude a MongoDB Atlas  ? lien
Creer un serveur MCP qui permet a Claude de lire/ecrire dans la base de donnees StatusPulse. | ~4h |
| Optionnel | Bootcamp Hackceleration (sessions live FR)  ? lien
Sessions live 18h30-20h ou 12h30-14h avec formateurs francophones. ~490 euros TTC. | ~2h |

Objectif fin juin  **Maitrise de l'API Claude + premier MCP fonctionnel sur StatusPulse**


**PHASE 3  Expert / Architecte CCA**     Juillet -> 31 aout  |  5-6h/semaine     ~35 heures

| Type | Module & Ressource | Dur?e |
|---|---|---|
| Lab | Claude Code 101 + Claude Code in Action  ? lien
Maitriser Claude Code dans terminal et VS Code. CLAUDE.md, subagents, hooks, MCP dans le workflow dev. | ~6h |
| Theorie | Agentic Architecture & Tool Use  ? lien
Workflows agentiques, multi-turn conversations, tool design, RAG (Retrieval Augmented Generation). Docs officielles. | ~5h |
| Lab | MCP Advanced + Subagents (Academy)  ? lien
Construire un orchestrateur multi-agents pour StatusPulse : analyse incidents + alertes automatiques. | ~6h |
| Theorie | Responsible AI ? Constitutional AI & Safety  ? lien
CAI, gestion des refus, red-teaming, compliance enterprise. Domaine requis pour la certification CCA-F. | ~2h |
| Projet | Projet Capstone : StatusPulse IA-powered
Integrer Claude dans toute l'app : analyse d'incidents, generation de rapports, alertes intelligentes. Architecture production-grade. | ~10h |
| Exam | Prepa & Examen CCA-F (Claude Certified Architect)  ? lien
629 questions pratiques. Inscription via Claude Partner Network (gratuit). Examen surveille, 5 domaines. | ~6h |

Objectif fin aout  **Certification CCA-F obtenue + StatusPulse IA-powered en production**
